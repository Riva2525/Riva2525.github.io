	<h1>Sistema de Reservas para Agência de Viagens</h1>
	<p>Um Sistema de Controle de Ordens de Serviço pode ser adaptado para uma Agência de Viagens, funcionando como um Sistema de Reservas.</p>
	<h2>Adaptação</h2>
	<p>Para adaptar o sistema, é necessário realizar as seguintes alterações:</p>
	<ol>
		<li>Cliente → Passageiro: Substituir a nomenclatura de "cliente" por "passageiro" em todo o sistema.</li>
		<li>Serviço → Reserva de Viagem: Substituir a nomenclatura de "serviço" por "reserva de viagem" em todo o sistema.</li>
		<li>Ordem de Serviço → Registro de Reserva: Substituir a nomenclatura de "ordem de serviço" por "registro de reserva" em todo o sistema.</li>
	</ol>
	<h2>Funcionalidades do Sistema</h2>
	<p>O sistema básico permitirá:</p>
	<ul>
		<li>Cadastrar passageiros: registrar informações dos passageiros, como nome, endereço, telefone, etc.</li>
		<li>Registrar reservas: registrar informações das reservas, como destino, data, tipo de acomodação, etc.</li>
		<li>Listar todas as reservas: visualizar todas as reservas registradas no sistema.</li>
	</ul>
